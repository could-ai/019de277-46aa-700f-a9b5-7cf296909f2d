# Программа поддержки от WB

Это приложение представляет собой карусель-статью о специальной программе поддержки производителей из России от Wildberries. Пользователь может пролистывать информационные слайды и в конце перейти по ссылке для подачи заявки на участие.

## Функционал приложения

- Интерактивная карусель (PageView) для пролистывания слайдов
- Плавная анимация индикатора страниц
- Темная тема с акцентным цветом бренда Wildberries
- Кнопка со ссылкой для перехода на официальный сайт программы (`url_launcher`)
- Отзывчивый интерфейс с крупной типографикой и иконками

## Основные пользовательские сценарии

1. При запуске приложения пользователь видит первый слайд с обзором программы поддержки от WB.
2. С помощью кнопок "Вперед" / "Назад" или свайпом пользователь перелистывает карточки (Кейсы, Для кого это, Дедлайн).
3. На последнем слайде ("Подача заявок") доступна кнопка "Перейти на сайт".
4. При нажатии на кнопку открывается системный браузер со страницей `https://rost.wb.ru/`.

## Технологический стек
